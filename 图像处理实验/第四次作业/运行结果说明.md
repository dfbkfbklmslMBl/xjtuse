# 运行结果说明

## 程序运行界面

运行`webcam_alexnet.m`程序后，将显示如下界面：

![程序运行界面示例](https://example.com/screenshot.png)

（上图为示例图，实际运行时将显示您摄像头采集的实时图像及分类结果）

## 界面组成部分

1. **主画面**：显示摄像头实时捕获的图像
2. **分类结果文本**：在图像上方显示当前识别的物体类别和置信度
3. **退出按钮**：位于右下角，用于停止程序

## 实际效果说明

1. 当摄像头对准不同物体时，分类结果会实时更新
2. AlexNet模型可以识别1000种不同类别的物体，包括：
   - 各种动物（猫、狗、鸟等）
   - 日常物品（电脑、手机、水杯等）
   - 食物（苹果、香蕉、披萨等）
   - 交通工具（汽车、自行车等）
   - 其他常见物体
3. 分类结果显示格式为：`分类结果: [类别名称] 置信度: [百分比]`
4. 置信度表示模型对该分类结果的确信程度，数值越高表示模型越确信

## 可能的分类效果示例

当摄像头对准键盘时，可能的分类结果：
```
分类结果: keyboard 置信度: 98.56%
```

当摄像头对准苹果时，可能的分类结果：
```
分类结果: apple 置信度: 87.23%
```

注意：实际分类结果将取决于图像质量、光线条件以及所对准物体的特征。为获得最佳效果，建议在光线充足的环境中使用程序。
