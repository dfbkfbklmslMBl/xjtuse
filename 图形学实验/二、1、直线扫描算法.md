### 二、1、直线扫描算法

#### DDA算法（Digital Differential Analyzer）
- 当直线斜率小于1时，从x的左端点开始，向x右端点步进，步长为1，以此计算出y的坐标，并取（x，int(y+0.5)）为当前点的坐标。当直线斜率绝对值大于1，以y轴作为步进方向

#### 中点画线法
假定直线斜率 k 在 0~1 之间，当前像素点为（xₚ，yₚ），则下一个像素点为 p₁（xₚ + 1，yₚ）或 p₂（xₚ + 1，yₚ + 1）。设 p₁和 p₂的中点为 M（xₚ + 1，yₚ + 0.5），Q 为理想直线和 x = xₚ + 1 的垂线的交点。当 M 在 Q 的上面时，取 p₁为下一像素点；反之，p₂为下一像素点。

#### Bresenham算法
- 过各行、各列像素中心构造一组虚拟网格线，按直线从起点到终点的顺序计算直线与各垂直网格线的交点，然后确定该列像素中与此交点最近的像素。

**Bresenham算法程序实现：**
```c
void Bresenhamline(int x0, int y0, int x1, int y1, int color) {
    int x, y, dx, dy, p;
    dx = x1 - x0; dy = y1 - y0;
    x = x0; y = y0;
    if (dy < 0) { dy = -dy; dx = -dx; }
    int i = 2 * dy - dx; p = 2 * dy;
    for (; x <= x1; x++) {
        drawpixel(x, y, color);
        if (i < 0) { i += 2 * dy; } else {
            if (i > 0) { y++; }
            i += 2 * (dy - dx);
        }
    }
}
```

### 二、4、字符串精度

字符串精度可以分为以下三个精度：
1. **串精度**：将包围字串的外接矩形对窗口作裁剪，当字符串方框整个在窗口内时予以显示，否则不显示。
2. **字符精度**：将包围字的外接矩形对窗口作裁剪，当字符方框整个在窗口内时予以显示，否则不显示。
3. **笔画/象素精度**：将矢量字符的笔划分解成直线段对窗口作裁剪；将点阵字符像素相对窗口边界作取舍，当字符方框整个在窗口内时予以显示，否则不显示。

### 二、5、字符类型及其特点

字符分为两种类型：
1. **点阵字符**：存储量大，易于表示。
2. **矢量字符**：存储量小，美观，变换方便；但需要光栅化（将几何数据经过一系列变换最终转换为像素）后才能显示。

### 二、8、走样和反走样

**走样**：由于显示器的空间分辨率有限，因像素逼近误差，使所画图形产生畸变（台阶、锯齿）的现象（用离散量表示连续量引起的失真）。

**反走样**：用于减少或消除这种效果的技术。

**反走样的主要方法**：
1. **提高分辨率**：显示器分辨率提高，直线经过的像素加倍，锯齿也加倍，同时每个阶梯的宽度减少，显示出来的直线就更加平滑。只能减轻而不能消除锯齿现象。
2. **区域采样**：假定每个像素是具有一定面积的小区域，将直线段看做具有一定宽度的狭长矩形，当直线段和像素有交时，求出两者的相交区域面积，然后根据相交区域面积的大小确定该像素的亮度值。
3. **加权区域采样**：在区域采样的基础上，使相交区域对像素亮度的贡献依赖于该区域与像素中心的距离。

### 三、5、Bezier曲线形状调整方法

设一条三次Bezier曲线的控制顶点为
$$
(P_0, P_1, P_2, P_3)
$$
。对曲线上一点\(P(0.5)\)，及一个给定的目标点\(T\)，给出一种调整Bezier曲线形状的方法，使得\(P(0.5))精确通过点T*T*。

$$
\begin{align}
&假设我们改变其中的一个控制顶点，比如把P₁改成(P₁ + λ)，使得P(0.5)精确通过点T，将修改后的曲线记为\\&(\widehat{P}(t))，则：\\
&[ \widehat{P}(t)|*{\frac{1}{2}} = \sum*{t = 0}^{3}P_iB_{i,3}(t)|*{\frac{1}{2}} + λB*{1,3}(\frac{1}{2}) ]\\
&即(T = P(\frac{1}{2}) + λB_{1,3}(\frac{1}{2}))，\\
&求得(λ = (T - P(\frac{1}{2})/B_{1,3}(\frac{1}{2})))，\\
&所以调整方法为：把(P₁)调整到(P₁ + (T - P(\frac{1}{2}))/B_{1,3}(\frac{1}{2}))。\end{align}
$$

### 三、14、形体表示方法

常见的形体表示方法有：

1. **线框模型**：只能表示一些简单模型。
2. **实体模型**：完整的、无歧义地表示三维形状，不适合物体表面的运算：关照计算、阴影计算和光线求交计算。
3. **表面模型**：基于连续参数曲面的表示方法（如样条等）和离散表面表示方法（如三角网格）。容易表达工业中较光滑的曲线曲面，难以表达真实世界的物体。
